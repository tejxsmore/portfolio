# Personal Portfolio Website

This is the repository for my personal portfolio website. The website is built using Astro 🚀, React, Typescript, TailwindCSS and is hosted on VERCEL - https://tejasmore.vercel.app

# Table of Contents

- [Installation](#installation)
- [Usage](#usage)

# Installation

To run the website locally, you'll need to clone th repository and type "npm run dev" in the terminal. Your site will be hosted locally on https://localhost:3000

# Usage

The website showcases my recent projects, my skills and my contact information. Visitors can download my Resume, read about my background, and get in touch with me through the contact form.
